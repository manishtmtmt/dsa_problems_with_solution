## **Masai Even Odd and Both**

You are given two numbers, stored in the variable with the following names
#
    one, two

If both the values are even, print `Even`

Else if both the values are odd, print `Odd`

else print `Even-Odd`

#### **Input Format**

-

#### **Output Format**

- 

#### **Sample Input**
    input: 3 6

#### **Sample Output**
    output: Even-Odd
